BlueML (Open Source)

[Jan 2018 - Present]

I researched features that can be interpreted from electrocardiogram (EKG) data to classify heart beats with machine learning. I also parsed the EKG files into bitstream and converted those into JSON formats to be sent over to the server. On the frontend side I also worked on the overall look and functionality of the web app

Worked with: Machine Learning Research, Agile Development, BootStrap, Javascript, HTML5, Angular, MongoDB, GitHub w/ ZenHub

SOCRAT (Open Source)

[Jan 2018 - Present]

I developed the visualization for the "Confidence Interval for One Proportion T Test". I also constructed the "Pilot Study" module used to help plan a pilot research study to estimate the expected error relative to the sample size

Worked with: Coffeescript, Vegalite, Jade, npm, Node, GitHub w/ Zenhub, Agile Development

  • GitHub

Kirigami Solar Cell Android App

[Jan 2017 - Dec 2017]

I constructed an Android App for an upcoming startup to use as an interface to control Kirigami Solar Cells through Firebase. The app has since been modified to communicate with an Arduino through Bluetooth and uses Location and Position Sensing ability (angle of phone). To communicate with the Solar Cells I coded an algorithm that parses byte arrays received from solar panel and converts it into readable Strings.

Worked with: Android App Development, Bluetooth, Firebase, Java, XML, Photoshop
