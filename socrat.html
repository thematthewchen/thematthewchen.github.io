Statistics Online Computational Resource Analytics Toolbox (SOCRAT)

Overview

When I joined SOCRAT back in Fall 2017, this open source project already had a well established team with a large variety of skilled people and it was these people that I wanted to work with to gain more web development experience visualizing data. A bunch of statistical modules were being developed and it was in there where I was able to grow.

My Contributions

I first had to familiarize myself with the codebase and in the process I was able to quickly learn webdev basics. Afterwards, I created the visualization of the One Proportion T Test using Vegalite, CoffeeScript, and Jade. I also worked with a teammate of mine to construct the Pilot Study module used in estimating the expected error SD (sigma) relative to the sample size (N). In addition to that I got exposure to Node and working with a large project.

Other Stuff

SOCRAT is a subteam under the Multidisciplinary Design Program Project "SOCR Big Data & Predictive Analysis" at the University of Michigan led by Professor Ivo Dinov. We coordinated tasks using GitHub with ZenHub and utilized an Agile Development style.
